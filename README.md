# Todo-App-CRUD

Flutter application for managing daily life tasks.

#### Flow Diagram ####

#### TODO App ####

- User can create tasks
- User can edit tasks
- User can delete tasks
- User can add time to tasks
- User can mark his tasks as pending or done
